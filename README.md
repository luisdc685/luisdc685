# 📚 Industrial Standard

⚙️ **IEC 61131-3**

Industrial PLC programming standard including:

- Ladder Diagram (LD)  
- Structured Text (ST)  
- Function Block Diagram (FBD)

Used for structured and modular industrial automation systems.
